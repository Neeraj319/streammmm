Welcome to Streammmm's documentation!
=====================================

A twitch like platform in `NestJs
<https://docs.nestjs.com/>`__ with `RTMP
<https://en.wikipedia.org/wiki/Real-Time_Messaging_Protocol>`__ for *real-time* video streaming.
Streammmm mainly consists of backend only for now which can be accessed on 
`localhost
<http://localhost:8000>`__.
